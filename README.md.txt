# Sistema Telefonia
**Autor:** _Wagner Bizarro_      
**Sobre:** Sistema com ferramentas para telefonia 
**Stack:** HTML/CSS/JS/PHP/BootStrap
**Data:** 02/09/2024      

**Sobre:**  
WebServer com as funcionalidades para apoiar o setor de telefonia da empresa.
Ferramentas de:
1- Consulta e envio de SMS.
2- Ativa mensagem de falha massiva no PABX. 
3- Consulta notas de atendimento dada para os atendentes.
4- Consulta se a ligação de ser feita usando CSP(Código de Seleção de Prestadora).
5- Abertura de incidentes.
